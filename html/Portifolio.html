PetShop CãoFortável

Esse projeto utiliza a sintaxe Razor e a linguaguem C# com o framework Asp.Net MVC e tem como objetivo gerenciar informações de cadastro e agendadmento de clientes e funcionários de um PetShop. Com conexexão com o banco de dados SQL, o projeto permite a realização de um CRUD, que permite ao usuário/cliente se cadastrar, editar seu cadastro e exclui-lo, além de fazer agendamentos. Já o usuário/funcionário pode realizar, editar e excluir agendamentos exibidos em uma tabela específica.
Para acessar a página do cliente basta realizar um cadastro no site. Para acessar a página do funcionário, é necessário inserir o login "adm@gmail.com" e a senha 123.

Jogo da memória Gravity Falls

Este é um projeto de jogo de memória simples que utiliza HTML, CSS e JavaScript. O jogo consiste em encontrar todas as cartas correspondentes em um conjunto de cartas viradas para baixo. O jogador pode virar duas cartas por vez e, se elas não correspondem, as cartas voltam para a posição original. Se as cartas correspondem, elas permanecem viradas para cima e o jogador pode continuar virando as cartas até todas estarem para cima.
O jogo apresenta uma contagem de tempo, que começa em zero e é incrementada a cada segundo durante o jogo. Quando o jogo terminar o tempo gasto será exibido. O jogo pode ser jogado quantas vezes quiser.

Fita Filmes

Este projeto é um site de busca de filmes que utiliza uma API de filmes. O site apresenta uma lista de filmes separador por "Em alta" e seus gêneros, onde mostra o nome e a nota do respectivo filme. O projeto foi desenvolvido utilizando HTML, CSS e JavaScript.
Dentro do site, o usuário pode pesquisar por filmes, ver a sinopse deles ao clicar no mesmo. O site também possui uma opção de mudança de tema escuro para modo claro.
